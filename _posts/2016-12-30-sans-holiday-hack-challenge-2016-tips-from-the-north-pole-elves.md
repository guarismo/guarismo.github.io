## Tips from the North Pole Elves

Hello again,

Long time no see.

I was working on the SANS Holiday Hack Challenge and at last I've submitted my write up today, although I couldn't find 2 NetWars coins.

I'll share my solutions once the challenge is over after January 4th.

For now, I want to share the tips that the Elves from the North Pole gave us to help with the challenges.

The Elves in the North Pole give a lot of information and tips about possible vulnerabilities that can be exploited and tools to use.

[![](/assets/images/me.JPG)](/assets/images/me.JPG)

Me!

### 路 Alabaster Snowball:

o Mentions JSON parameter editing

o Recommends using BurpSuite

[![](/assets/images/JSON-Cheat_dungeon.jpg)](/assets/images/JSON-Cheat_dungeon.jpg)

### 路 Holly Evergreen:

o Once all the Cranberry Pi pieces are found she provides the Cranbian image.

[![](/assets/images/cranberry-pi - holly evergreen.JPG)](/assets/images/cranberry-pi - holly evergreen.JPG)

### 路 Shiny Upatree:

o Mentions APK files are just ZIP files

o Recommends JadX-gui to decompile APK

o Provides a link to download Joshua Wrights [presentation](https://pen-testing.sans.org/blog/2016/12/05/ghost-in-the-droid-reverse-engineering-android-apps) from HackFest 2016 on using Android Studio and JadX

[![](/assets/images/JadX for PAK - Shinny Upatree.jpg)](/assets/images/JadX for PAK - Shinny Upatree.jpg)

### 路 Pepper Minstix:

o Talks about the Meteor Framework and provides a link to the projects web site.

o Provides a link to Tim Medins [blog post](https://pen-testing.sans.org/blog/2016/12/06/mining-meteor) about using Tampermonkey and his own Meteor Miner JavaScript to exploit over subscription sharing too much data with Meteor.

o Provides a link to download an old version of the Dungeon game.

[![](/assets/images/dungeon-zip - Pepper Minstix.jpg)](/assets/images/dungeon-zip - Pepper Minstix.jpg)

### 路 Minty Candycane:

o Mentions NMAP and how the default sC switch really works well.

o Mentions John the Ripper and provides a link to download the [Rockyou](https://wiki.skullsecurity.org/index.php?title=Passwords) password database for brute forcing.

[![](/assets/images/NMAP-John-Rockyou - Minty Candycane.jpg)](/assets/images/NMAP-John-Rockyou - Minty Candycane.jpg)

### 路 Sugarplum Mary:

o Mentions how PHP Filters can be used to read all kind of I/O streams.

o Provides a link to Jeff McJunkis [blog post](https://pen-testing.sans.org/blog/2016/12/07/getting-moar-value-out-of-php-local-file-include-vulnerabilities) about exploiting local file inclusion with php filters. 

[![](/assets/images/PHP-clue - Sugarplum Mary.jpg)](/assets/images/PHP-clue - Sugarplum Mary.jpg)

### 路 Bushy Evergreen:

o Mentions apktool to decompile Android apps.

o Says JadX is great to look at the code but not good for recompiling, but with Apktool, I can modify the smali code and recompile.

o Reminds to signed recompiled APK files before installing.

o Provides a link to Joshua Wrights [video](https://www.youtube.com/watch?v=mo2yZVRicW0) about manipulating APK files with apktool.

[![](/assets/images/apktool - Bushy Evergreen.jpg)](/assets/images/apktool - Bushy Evergreen.jpg)

### 路 Wunorse Openslae:

o Mentions how hard its been to manage so many SD cards.

oProvides link to Joshua Wrights [blog post](https://pen-testing.sans.org/blog/2016/12/07/mount-a-raspberry-pi-file-system-image) about managing SD card images and how to mount them

[![](/assets/images/mounting-SD - Wunorse Openslae.JPG)](/assets/images/mounting-SD - Wunorse Openslae.JPG)

The same tips and others are provided in the SantaGram app once you manage to install it and look at the post from elves.
